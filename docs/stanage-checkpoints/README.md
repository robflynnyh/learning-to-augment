# Stanage Checkpoint Notes

Readable docs for the Learn-to-Augment checkpoints discovered on Stanage.

Start here:

- [`CHECKPOINT_SUMMARY_STANAGE.md`](CHECKPOINT_SUMMARY_STANAGE.md) — human-readable experiment story and checkpoint-family summary.
- [`UFMR_VS_CFM.md`](UFMR_VS_CFM.md) — focused comparison of UFMR and its audio-conditioned analogue CFM, including training/eval commands.
- [`EVALUATION_COMMANDS_STANAGE.md`](EVALUATION_COMMANDS_STANAGE.md) — launch commands and eval config mapping for checkpoint families.
- [`CHECKPOINT_MANIFEST_STANAGE.md`](CHECKPOINT_MANIFEST_STANAGE.md) — detailed checkpoint-by-checkpoint manifest extracted from embedded configs.

Raw inspection artifacts currently live at the repo root as hidden files plus `inspect_l2a_ckpts.py`; they are kept separately for traceability.
